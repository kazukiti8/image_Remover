## アプリケーションの起動

コマンドラインから以下のコマンドを実行します：

```bash
python main_application.py
```

## 基本操作

### ディレクトリの選択

1. **メニューから選択**: 「ファイル」→「ディレクトリを開く」を選択します。
2. **ドラッグ&ドロップ**: フォルダをアプリケーションウィンドウにドラッグ&ドロップします。
3. **ツールバー**: ツールバーの「開く」ボタンをクリックします。

### 画像の閲覧

- **サムネイル表示**: 左側のグリッドに画像のサムネイルが表示されます。
- **サイズ変更**: 「表示」メニューの「サムネイルサイズ」から選択するか、サムネイルビュー上部のスライダーを使用します。
- **並べ替え**: サムネイルビュー上部の「並び替え」ドロップダウンで名前、日付、サイズ順に並べ替えられます。

### 画像の詳細表示

- **プレビュー**: サムネイルをクリックすると、右側に大きなプレビューが表示されます。
- **拡大/縮小**: プレビュー上でマウスホイールを使うか、ツールバーのズームコントロールを使用します。
- **ドラッグ移動**: プレビュー上で左クリックしたままドラッグすると表示位置を移動できます。

### 画像の選択

- **単一選択**: サムネイルをクリックします。
- **複数選択**: サムネイルのチェックボックスをクリックします。
- **すべて選択/解除**: 「すべて選択」または「選択解除」ボタンをクリックします。

### 画像の操作

- **削除**: 選択した画像を削除するには「選択した画像を削除」ボタンをクリックします。
- **移動**: 選択した画像を移動するには「選択した画像を移動」ボタンをクリックします。

## 画像検出機能

### ブレている画像の検出

ブレている画像を自動検出します：

1. 「検出」メニューから「ブレている画像を検出」を選択します。
2. 検出が完了すると、ブレている画像のサムネイルが選択状態になります。
3. 確認ダイアログで「はい」を選択すると、検出された画像が自動的に選択されます。

### 類似画像の検出

視覚的に類似した画像を検出します：

1. 「検出」メニューから「類似画像を検出」を選択します。
2. 検出が完了すると、類似画像のサムネイルが選択状態になります。
3. 確認ダイアログで「はい」を選択すると、検出された画像が自動的に選択されます。

### 重複画像の検出

完全に同一の画像を検出します：

1. 「検出」メニューから「重複画像を検出」を選択します。
2. 検出が完了すると、重複画像のサムネイルが選択状態になります。
3. 確認ダイアログで「はい」を選択すると、検出された画像が自動的に選択されます。

### 全ての検出の実行

全ての検出処理をまとめて実行します：

1. 「検出」メニューから「全ての検出を実行」を選択します。
2. 検出が完了すると、検出された全ての画像のサムネイルが選択状態になります。

## 画質評価機能

AIによる画像品質評価を実行します：

1. 「評価」メニューから「画質評価を実行」を選択するか、ツールバーの「画質評価」ボタンをクリックします。
2. 「画質評価」タブが表示され、評価が開始されます。
3. 評価が完了すると、評価結果の概要が表示されます。
4. 各画像を選択すると、詳細な評価結果が表示されます。

### 評価項目

- **シャープネス**: 画像のシャープさ（ブレていないか）
- **露出**: 適切な明るさか
- **コントラスト**: 明暗の差が適切か
- **ノイズ**: 画像にノイズがないか
- **構図**: 画像の構図が良いか

各項目は0〜10点で評価され、総合スコアも算出されます。

## バッチ処理

複数のディレクトリを一括で処理します：

1. 「ファイル」メニューから「バッチ処理」を選択します。
2. 「バッチ処理」ドックウィンドウが表示されます。
3. 「ディレクトリ追加」ボタンで処理対象ディレクトリを追加します。
4. 処理タイプ（クリーンアップ/画質評価）を選択します。
5. 「バッチ処理開始」ボタンをクリックして処理を開始します。
6. 処理が完了すると、結果のサマリーが表示されます。
7. 「結果を確認」ボタンで詳細な結果を確認できます。

## 設定のカスタマイズ

アプリケーションの設定をカスタマイズします：

1. 「編集」メニューから「設定」を選択するか、ツールバーの「設定」ボタンをクリックします。
2. 設定ダイアログが表示され、以下のタブで設定を変更できます：
   - **一般**: 基本設定
   - **クリーンアップ**: 検出閾値などの設定
   - **画質評価**: 評価の重みづけ設定
   - **表示**: UI表示設定

### 主な設定項目

#### 一般設定
- **最後に使用したディレクトリ**: 起動時に開くディレクトリ
- **プレビューを自動的にフィット**: プレビューをウィンドウサイズに合わせる
- **テーマ**: UI表示テーマ
- **言語**: インターフェース言語
- **削除時に確認**: 削除前に確認ダイアログを表示するか

#### クリーンアップ設定
- **ブレ検出閾値**: 値が低いほど厳格に検出（デフォルト: 100.0）
- **類似画像閾値**: 値が低いほど厳格に検出（デフォルト: 10）
- **検出機能の有効/無効**: 各検出機能の有効/無効を切り替え

#### 画質評価設定
- **各評価項目の重み**: 各評価項目の重要度を設定

#### 表示設定
- **サムネイルサイズ**: サムネイルの表示サイズ
- **グリッド列数**: サムネイル表示の列数
- **EXIF表示**: EXIF情報の表示/非表示
- **ウィンドウサイズ**: デフォルトのウィンドウサイズ

### 設定のインポート/エクスポート

- **エクスポート**: 現在の設定をJSONファイルに保存します
- **インポート**: JSONファイルから設定を読み込みます

## トラブルシューティング

### 一般的な問題

#### 画像が表示されない
- 画像フォーマットがサポートされているか確認してください（JPEG, PNG, BMP, GIF, WEBP）
- ファイルが破損していないか確認してください

#### 処理が遅い
- 大量の画像がある場合は処理に時間がかかることがあります
- クリーンアップの閾値を調整して精度と速度のバランスを取ってください

#### エラーメッセージが表示される
- アプリケーションを再起動してみてください
- コンソール出力にエラーメッセージがないか確認してください

### 具体的な問題と解決策

#### 「ブレている画像の検出に失敗しました」エラー
- ブレ検出閾値を調整してみてください（設定→クリーンアップ→ブレ検出閾値）
- 非常に大きなサイズの画像が含まれていないか確認してください

#### 「類似画像の検出に失敗しました」エラー
- 類似画像閾値を調整してみてください（設定→クリーンアップ→類似画像閾値）
- メモリ不足の可能性があります。不要なアプリケーションを閉じてみてください

#### バッチ処理が途中で止まる
- 処理対象のディレクトリ数を減らしてみてください
- 一度にすべての検出を実行せず、個別に実行してみてください

これらの解決策で問題が解決しない場合は、アプリケーションを再インストールするか、開発者に問い合わせてください。
